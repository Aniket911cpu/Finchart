## Conversion Funnel Strategy

1. **Sign-up hook:** Email-only, no credit card
2. **Aha moment:** User creates first alert → triggers → sees magic
3. **Paywall triggers:**
   - Adding 4th watchlist → "Upgrade to Pro"
   - Exceeding 10 alerts → Paywall modal
   - Accessing historical data > 1yr → Upgrade CTA
   - Screener with >3 filters → Upgrade CTA
4. **Nurture:** 7-email onboarding sequence, highlight Pro features
5. **Discount:** 30-day Pro trial, annual plan 31% savings
